Report security vulnerabilities to the maintainers.

If you discover a vulnerability, please:

1. Do not create a public issue. Instead contact the maintainers via email: security@your-org.example (replace with real address).
2. Provide a short summary, reproduction steps, and a suggested fix if possible.
3. The maintainers will respond and coordinate a disclosure timeline.

This repository also includes CodeQL analysis and Dependabot configuration to help surface issues.
