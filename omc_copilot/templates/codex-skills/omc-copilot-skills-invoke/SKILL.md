# OMC Skills Invoke

Codex-native OMC skills are installed by:

```bash
omc-copilot setup --runtime codex --target <path>
```

For implemented workflows, prefer the matching `omc-copilot` command with `--runtime codex`. If a requested OMC skill is metadata-only, explain the current limitation and identify the nearest implemented command.
